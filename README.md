# 📚 Taller 1: Programación Funcional y Persistencia en Elixir

Este repositorio tiene la solución a los ejercicios propuestos para el taller de **Programación II** en la **Universidad del Quindío**. 
El proyecto está estructurado en dos partes principales los cuales son los dos puntos propuestos.

---

## 📂 Estructura del Repositorio

El repositorio se organiza de la siguiente manera:

*   **`/punto_1_Gimnasio`**: 
*   **`/punto_2_Inventario`**: 

Cada carpeta contiene su propio archivo `README.md` con las instrucciones específicas de ejecución y detalles técnicos.

---

## 🎯 Objetivos del Taller
*   Implementar estructuras de datos mediante `Structs` y `Mapas`.
*   Garantizar la persistencia de información en formato **JSON**.
*   Aplicar técnicas de **Pattern Matching** y recursividad.
*   Documentar el proceso de desarrollo y el uso ético de herramientas de **IA**.


---

## 🛠️ Tecnologías Globales
*   **Elixir** 
*   **Mix**
*   **Git/GitHub** 

---

## 🤖 Uso de Inteligencia Artificial
Para la realización de este taller se utilizó **Gemini** como asistente de desarrollo. Se empleó principalmente para:
*   Depuración de errores de tipos (Átomos vs Strings) en la carga de archivos.
*   Configuración de entornos en Windows PowerShell.
*   Optimización de la recursividad en los menús de consola.
